# Dengue Prototype Demo

React + Tailwind interactive prototype for thesis presentation.

## What this is
- Non-functional demo prototype
- Interactive route-based screens for presentation
- Matches the clean premium prototype style discussed in the deck

## Screens included
- Login
- Dashboard
- Upload
- Forecast
- Map
- Reports

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
This prototype is for presentation and demo purposes only. It does not connect to a backend or real datasets.
